# Stage-M1-IGE-
Voici les notebooks que j'ai ecrit durant mon stage de M1 sur les couvertures neigeuses dans les Alpes, le travail a ete realise sous dahu (Gricad, universite Grenoble Alpes) ou l'ensemble des documents utilise sont disponibles sur bettik 
 Les notebooks "Comparaison..." ont pour de comparer differentes banques de donnees SNOW-CCI. 
 Le Notebook "snowcover..." a pour but d'etablir une comparasaison entre ces donnees avec un Modele (MAR ERA 5) et d'etablir la meilleure facon d'obtenir une SCF (Snow Cover Fraction) a partir d'une hauteur de neige.
 Le notebook "Corrrection..." a lui pour objectif d'esssayer plusierus technique de corretion du modele par rapport aux donnnes (les meilleures connsiderees dans les notebooks de comparaison cad v4 gap-filled) et d'etablir des tendance de ligne de neige avec une version corrigee du modele.
